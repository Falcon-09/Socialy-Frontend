## üöÄ Features
- Login/Signup User Account
- Login using Email/Username
- Email verification via Otp
- Update Profile/Password User Account
- Infinite Scroll Posts
- Suggested Users
- Follow/Unfollow Users
- Like/Unlike Posts
- Comment on Post
- Save/Unsave Post
- Share New Post
- Delete Own Posts
- Add New Chat
- Message Users
- Online User Status
- See Followers/Following of Users
- Post Liked by Users
- Emoji Enabled
